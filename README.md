# Andy Gonzales - Portafolio Personal

Este es el repositorio de mi portafolio web personal, donde presento mi perfil profesional, mis proyectos más destacados, tecnologías que manejo y mi formación como **Desarrollador Backend**, especializado en **Java, Spring Boot y bases de datos**, con conocimientos sólidos en **React**.

## 🚀 Tecnologías utilizadas

Este proyecto ha sido desarrollado con un ecosistema frontend moderno, enfocándose fuertemente en el rendimiento, la accesibilidad y un excelente diseño UI/UX.

### Core
- **[React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)**: Motor principal e interactividad con tipado estricto.
- **[Vite 8](https://vitejs.dev/)**: Empaquetador ultrarrápido para una excelente experiencia de desarrollo.

### Diseño y UI
- **[Tailwind CSS v4](https://tailwindcss.com/)**: Motor de estilos y utilidades CSS responsivas.
- **[shadcn/ui](https://ui.shadcn.com/)**: Colección de componentes de interfaz accesibles y personalizables (usando `clsx`, `tailwind-merge` y `class-variance-authority`).
- **[Base UI](https://base-ui.com/)**: Primitivas accesibles y sin estilos para componentes interactivos.
- **[Lucide React](https://lucide.dev/)**: Iconografía hermosa y consistente.
- **[Geist Font](https://vercel.com/font)**: Tipografía principal optimizada para interfaces de usuario.
- **[tw-animate-css](https://github.com/)**: Utilidades de animación para Tailwind.
